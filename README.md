# hydrogen-pass 密码管理

本项目是基于 JavaFX 的桌面密码管理工具。

1. 适合于管理一个服务器的多个账号；
1. 可以用来保存运维笔记，方便运维。

本项目依赖 [hydrogen-fx](https://gitee.com/yidinghe/hydrogen-fx) 项目，请先将该项目编译并安装到本地 Maven 库。

Icons of this app come from [icons8](https://icons8.com/).
